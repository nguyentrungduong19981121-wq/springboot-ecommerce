## üéØ API Endpoints Detail

### 1. GET /api/products
**Features**:
- ‚úÖ Get all products
- ‚úÖ Search by name: `?name=laptop`
- ‚úÖ Filter by category: `?categoryId=1`
- ‚úÖ Sort by price: `?sortBy=price_asc` or `?sortBy=price_desc`
- ‚úÖ Combined filters: `?name=phone&categoryId=2&sortBy=price_desc`

**Example**:
```bash
curl "http://localhost:8080/api/products?name=iphone&sortBy=price_asc"
```

### 2. GET /api/products/{id}
**Features**:
- ‚úÖ Get product details by ID
- ‚úÖ Includes category info
- ‚úÖ Includes all image URLs
- ‚úÖ 404 error if not found

**Example**:
```bash
curl http://localhost:8080/api/products/1
```

### 3. POST /api/products
**Features**:
- ‚úÖ Create new product
- ‚úÖ Validate unique slug
- ‚úÖ Validate required fields
- ‚úÖ Validate price >= 0
- ‚úÖ Validate stock >= 0
- ‚úÖ Optional category assignment
- ‚úÖ Optional multiple images
- ‚úÖ Returns 201 Created

**Example**:
```bash
curl -X POST http://localhost:8080/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "New Product",
    "slug": "new-product",
    "price": 1000.0,
    "stock": 10,
    "categoryId": 1,
    "imageUrls": ["https://example.com/image1.jpg"]
  }'
```

### 4. PUT /api/products/{id}
**Features**:
- ‚úÖ Partial update (only provided fields)
- ‚úÖ Validate unique slug if changed
- ‚úÖ Validate category exists if provided
- ‚úÖ Replace all images if imageUrls provided
- ‚úÖ 404 if product not found

**Example**:
```bash
curl -X PUT http://localhost:8080/api/products/1 \
  -H "Content-Type: application/json" \
  -d '{"price": 2000.0, "stock": 5}'
```

### 5. DELETE /api/products/{id}
**Features**:
- ‚úÖ Delete product
- ‚úÖ Cascade delete images
- ‚úÖ Returns 204 No Content
- ‚úÖ 404 if product not found

**Example**:
```bash
curl -X DELETE http://localhost:8080/api/products/1
```
